\# Low-Fidelity Wireframes



This folder contains the low-fidelity wireframes for the hardware talent platform.  

The wireframes illustrate the core user flow from discovery to engagement, focusing on clarity, trust, and high-confidence decision making rather than visual styling.



The key screens include:

\- Landing / Search Entry

\- Manual \& AI Search Results

\- Profile Detail (Freelancer / Cluster)

\- Contact \& Message
